Enum TimeWeighting

Defines the weighting of samples within when applying average based aggregation functions.

Namespace: Hsm.DataAnalysisService.QueryLanguage.Request
Assembly: Hsm.DataAnalysisService.QueryLanguage.dll
Syntax
public enum TimeWeighting

Fields

Name Description
Equal

All samples are equally weighted.
HalfTimePreviousNext

The weighting of a sample is the half time to the previous plus the half time to the next sample. The first sample is weighted with half the time to the start of the analysis interval. The last sample is weighted with half the time to the end of the analysis interval.
TimeToNext

The weighting of a sample is the timespan to the next sample. Last sample in a series of samples is weighted with the timespan to the end of the analysis interval.
TimeToPrevious

The weighting of a sample is the timespan to the previous sample. the First sample in a series of samples is weighted with the timespan to the start of the analysis interval.
